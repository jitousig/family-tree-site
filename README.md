# The Descent of Tousignant-Barnes

**[View the tree →](https://jitousig.github.io/family-tree-site/)**

An interactive family tree and written narrative tracing the ancestry of Joel & Stephen
Tousignant-Barnes of Calgary — maternal French-Canadian and Irish lines (Tousignant,
Tétrault/Barré, Heffernan, Charlebois) and paternal British ones (Barnes/Douglas of Scotland and
Cumberland, Buckerfield/Henchman/Dowler of England), converging on Kneehill County, Alberta.
Roughly 80 ancestor records across 12 generations, each graded **sourced**, **reported**, or
**gap** according to the evidence behind it.

The page is a single self-contained HTML file. Views: tree, fan chart, migration map, timeline,
research frontier, surname index, stories, and the narrative.

---

*This repository is generated — it holds only the built page and the narrative PDF. The research
project that produces it (the source dossier, the data layer, and the primary-source record images)
lives in a separate private repository. Do not edit `index.html` here; it is overwritten on every
